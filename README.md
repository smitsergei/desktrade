# Еженедельник трейдера

Веб-приложение для трейдинга корпоративной отчетности на предикшен рынках.

## Особенности

- **Еженедельное планирование** с разделами "До открытия" и "На закрытии"
- **Риск-менеджмент** с автоматическим расчетом размера позиции
- **Отслеживание P&L** с историей баланса
- **Корпоративная отчетность** - специализация под предикшен рынки

## Технологии

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Prisma ORM
- PostgreSQL (Neon.tech)
- NextAuth.js

## Установка

1. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd desktrade
```

2. Установите зависимости:
```bash
npm install
```

3. Настройте переменные окружения в `.env.local`:
```env
# Database
DATABASE_URL="postgresql://username:password@host:5432/database?sslmode=require"

# NextAuth
NEXTAUTH_URL="http://localhost:3000"
NEXTAUTH_SECRET="your-secret-key-here"
```

4. Сгенерируйте Prisma клиент:
```bash
npx prisma generate
```

5. Запустите миграции базы данных:
```bash
npx prisma db push
```

6. Запустите приложение:
```bash
npm run dev
```

## Использование

1. Войдите в систему или создайте аккаунт
2. Настройте депозит и риски в разделе настроек
3. Добавляйте предикшены для каждого дня:
   - Укажите тикер
   - Выберите рейтинг (1-3)
   - Установите цену предикшена (0-1)
   - Укажите уверенность в %
4. Закрывайте сделки при получении результата
5. Следите за P&L и историей баланса

## Логика расчета

- **Размер позиции**: (Депозит × Риск%) / Цена предикшена
- **При выигрыше**: +Размер позиции
- **При проигрыше**: -Размер позиции × Цена предикшена

## Деплой

Проект готов к деплою на Vercel. Убедитесь, что переменные окружения настроены в Vercel Dashboard.
